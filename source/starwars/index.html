Star Wars Intro
This is a Star Wars title crawl inspired about page. It uses mostly CSS and a bit of Javascript to get things started when the user clicks for the first time.

The code can be seen in the source or on my github page.

This was largely an exercise in CSS and also a fun idea for a 'about' page or an introduction page to use on a website. I wasn't sure what the best way to end the crawl was so I just have it repeating in a loop, but that can be easily changed to something else. If this were a real 'about' page maybe a picture or link or something could appear at the end of the crawl.
